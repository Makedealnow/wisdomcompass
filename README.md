# WisdomCompass Website

Complete static website package for WisdomCompass.net.

## Upload to GitHub
Upload all files in this folder to a new GitHub repository.

## Deploy to Vercel
Import the GitHub repository into Vercel and deploy.

## Google Analytics
Replace G-XXXXXXXXXX in all HTML files with your Google Analytics Measurement ID.

## Google Search Console
Replace GOOGLE_SITE_VERIFICATION_CODE_HERE in all HTML files with your verification code.

## Sitemap
Submit this URL in Google Search Console:
https://wisdomcompass.net/sitemap.xml
