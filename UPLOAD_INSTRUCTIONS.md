# Upload Instructions

This ZIP is cleaned and flattened.

1. Extract this ZIP on your computer.
2. Upload the files and folders INSIDE the extracted folder to the GitHub wisdomcompass repository root.
3. Do NOT upload this ZIP file itself.
4. In GitHub, delete any old folder named wisdomcompass-main and any old .zip files.
5. Commit changes.
6. Let Vercel redeploy.

Required root items include:
- index.html
- script.js
- styles.css
- package.json
- api/guidance.js
- assets/
